# CloudnetPy 中文文档 (docs_zh)

本目录包含 CloudnetPy 项目的完整中文文档。

## 快速开始

### 1. 安装依赖

```bash
cd docs_zh
pip install -r requirements.txt
```

### 2. 构建文档

**Linux/macOS:**
```bash
make html
```

**Windows:**
```cmd
make.bat html
```

### 3. 查看文档

在浏览器中打开 `docs_zh/html/index.html` 或 `docs_zh/index.html`

## 文档内容

本中文文档包含以下内容的完整翻译：

- **概述** (overview.rst) - CloudnetPy 项目简介和 Cloudnet 处理方案说明
- **安装说明** (installation.rst) - 详细的安装步骤
- **快速入门** (quickstart.rst) - 从原始数据到产品生成的完整教程
- **API 参考** (api.rst) - 所有函数和类的详细文档
- **文件格式** (fileformat.rst) - 数据文件格式说明
- **模式评估** (model-evaluation/overview.rst) - 模式评估功能说明
- **用户指南** (guide.rst) - 用户使用指南
